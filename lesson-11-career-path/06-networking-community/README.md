### Conference Engagement Tips

```
┌─────────────────────────────────────────────────────────────┐
│              Conference Engagement Strategy                  │
│                                                              │
│  BEFORE                                                     │
│  • Review schedule and plan sessions                        │
│  • Research speakers you want to meet                       │
│  • Set networking goals (e.g., 5 meaningful connections)   │
│  • Prepare your elevator pitch                              │
│                                                              │
│  DURING                                                     │
│  • Attend hallway track (conversations between sessions)   │
│  • Ask questions after talks                                │
│  • Exchange contact info with people you connect with       │
│  • Take notes on interesting ideas                          │
│  • Share insights on social media (with speaker tags)       │
│                                                              │
│  AFTER                                                      │
│  • Connect on LinkedIn within 24-48 hours                   │
│  • Send personalized follow-up messages                     │
│  • Share key learnings with your team                       │
│  • Consider submitting CFP for next year                    │
└─────────────────────────────────────────────────────────────┘
```

## 6.3 Mentorship

### Finding a Mentor

| Approach | How |
|----------|-----|
| **Internal** | Ask someone 1-2 levels senior in your organization |
| **Professional network** | Reach out to connections you admire |
| **Conferences** | Follow up with speakers whose work resonates |
| **Online** | Engage with thought leaders' content |
| **Formal programs** | Company mentorship programs, professional associations |

### The Ask

```
┌─────────────────────────────────────────────────────────────┐
│              Sample Mentorship Request                       │
│                                                              │
│  "Hi [Name],                                                │
│                                                              │
│  I've been following your work on [specific topic] and     │
│  found your [article/talk/project] incredibly insightful.  │
│                                                              │
│  I'm currently a [role] working on [focus area], and I'm   │
│  looking to grow into [goal]. I noticed you've navigated   │
│  a similar path.                                            │
│                                                              │
│  Would you be open to a 30-minute conversation about       │
│  [specific topic]? I'd love to learn from your experience. │
│                                                              │
│  I'm happy to work around your schedule. Thank you for     │
│  considering this.                                          │
│                                                              │
│  Best,                                                      │
│  [Your name]"                                               │
└─────────────────────────────────────────────────────────────┘
```

### Being a Mentor

| Benefit | Description |
|---------|-------------|
| **Deepens knowledge** | Teaching clarifies your own understanding |
| **Builds leadership** | Practice coaching and developing others |
| **Expands network** | Your mentees become future colleagues |
| **Gives back** | Pay forward the help you received |
| **Fresh perspectives** | Learn from mentees' questions and approaches |

## 6.4 Building Your Network

### Networking Philosophy

```
┌─────────────────────────────────────────────────────────────┐
│              Effective Networking Mindset                    │
│                                                              │
│  GIVE FIRST                                                 │
│  "How can I help you?" before "What can you do for me?"   │
│  • Share knowledge freely                                   │
│  • Make introductions                                       │
│  • Offer feedback and support                               │
│  • Celebrate others' successes                              │
│                                                              │
│  BE AUTHENTIC                                               │
│  Build genuine relationships, not transactional contacts   │
│  • Be genuinely curious about others                        │
│  • Share your real experiences (including failures)        │
│  • Don't pretend to know things you don't                  │
│  • Follow up because you care, not because you need        │
│                                                              │
│  PLAY THE LONG GAME                                         │
│  Networks compound over time                                │
│  • Maintain relationships even when you don't need them    │
│  • Invest in relationships before you need help            │
│  • Small consistent touchpoints > occasional intense ones  │
└─────────────────────────────────────────────────────────────┘
```

### Relationship Maintenance

| Frequency | Activities |
|-----------|------------|
| **Weekly** | Engage with connections' content on social media |
| **Monthly** | Reach out to a few connections with relevant articles or updates |
| **Quarterly** | Coffee/video calls with close connections |
| **Annually** | Review your network and reconnect with dormant relationships |
