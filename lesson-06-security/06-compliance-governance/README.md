## Overview

Compliance and Governance covers regulatory requirements, data classification schemes, and incident response procedures.

## Key Concepts

### Regulatory Frameworks

| Framework | Focus |
|-----------|-------|
| GDPR | Data Privacy (EU) |
| HIPAA | Healthcare (US) |
| PCI DSS | Payment Cards |
| SOC 2 | Service Organizations |
| ISO 27001 | Information Security |

### Incident Response Phases

1. Preparation
2. Detection
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned
